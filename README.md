# dep-tracker

依赖分析工具，用于分析某个入口的所有依赖，以及每个依赖是怎么来的（寻找其依赖方）

## 使用方法

```
# 安装依赖
yarn

# 构建前端
yarn build

# 启动服务
yarn start
```

访问 http://localhost:3000/
